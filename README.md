# Smart Expense & Budget Tracker (with Dashboard)

## 📋 Project Overview
Managing personal finances effectively requires clear visibility. This project is a comprehensive Financial Tracker developed in Google Sheets that allows users to log expenses, set monthly budgets, and visualize their spending habits through an interactive dashboard.

## 🛠 Key Features
- **Dynamic Dashboard:** Visual representation of spending patterns using Pie charts, Bar graphs, and Scorecards.
- **Budget vs. Actual Analysis:** Automatically compares planned budgets with actual spending to highlight overspending.
- **Categorized Tracking:** Easily categorize expenses (e.g., Food, Rent, Entertainment) for better insights.
- **Real-Time Calculations:** Automatically updates total balance, total savings, and remaining budget as data is entered.
- **Monthly Summary:** A clean, automated overview of financial health at a glance.

## 🧰 Tech & Tools Used
- **Google Sheets:** Core platform for financial data logging.
- **Advanced Formulas:** Used `SUMIF`, `COUNTIF`, `IFERROR`, and complex arithmetic for financial calculations.
- **Data Visualization:** Integrated Charts and Graphs to create the dashboard.
- **Conditional Formatting:** Used to highlight budget alerts (e.g., turns Red when the budget is exceeded).
